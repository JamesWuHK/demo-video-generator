## Logo

项目 Logo 使用 🎬 (电影摄像机) 表情符号作为识别标志。

### 配色方案

## Demo 视频大纲

**时长：90-120 秒**

1. **开场 (0-10s)**
   - 项目标题和 tagline
   - 问题陈述：制作演示视频费时费力

2. **解决方案 (10-30s)**
   - 展示编写 YAML 分镜脚本
   - 运行 demovideo 命令
   - 自动生成视频

3. **核心功能 (30-70s)**
   - AI 语音合成
   - 自动浏览器录制
   - 精准声画同步
   - 字幕生成

4. **使用场景 (70-90s)**
   - 产品演示
   - 教程制作
   - 营销视频

5. **号召行动 (90-120s)**
   - GitHub 链接
   - 开源免费
   - 立即试用

## Logo 设计建议

由于本项目是开源工具，建议：
- 保持简洁专业
- 使用 🎬 emoji 或相关图标
- 配色符合技术工具风格
- 可以使用在线工具如 Canva 或 Figma 设计

## 创建资源的工具

- **Logo**: Canva, Figma, Adobe Illustrator
- **截图**: macOS Screenshot, Snagit, CloudApp
- **GIF**: Kap (macOS), LICEcap, ScreenToGif
- **视频**: OBS Studio, Loom, ScreenFlow
- **图片编辑**: GIMP, Photoshop, Figma

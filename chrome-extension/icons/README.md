# Chrome Extension Icons

由于无法直接生成图片文件，请使用以下方法创建图标：

## 方法 1：使用在线工具
访问 https://www.favicon-generator.org/ 上传一个图标图片（推荐使用 🎬 emoji 截图），生成 16x16, 48x48, 128x128 三个尺寸。

## 方法 2：使用现有图标
将任意 PNG 图标重命名为 icon16.png, icon48.png, icon128.png 并放入此目录。

## 方法 3：临时使用占位符
可以暂时不使用图标，在 manifest.json 中删除 icons 相关配置也能正常工作。
